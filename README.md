# my-first-pr

A practice repository for learning GitHub Pull Requests.

## About

This repository is used to practice the GitHub workflow:
1. Create a branch
2. Make changes
3. Open a Pull Request
4. Merge

## Getting Started

Clone the repository and start experimenting!

```bash
git clone https://github.com/R-Suzuki0806/my-first-pr.git
cd my-first-pr
```

## Contributing

1. Fork the repository
2. Create a new branch (`git checkout -b feature/your-feature`)
3. Commit your changes (`git commit -m 'Add your feature'`)
4. Push to the branch (`git push origin feature/your-feature`)
5. Open a Pull Request
